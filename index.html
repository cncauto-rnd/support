Calibration Factor Calculator

This widget takes four inputs and calculates the calibration factor for the FoldSafe controller.

FoldSafe Diagnostic page can be used to read the raw encoder count for this calculation. Measure the initial distance between tip of the tool and the top of the die using a measuring device and enter the reading .
